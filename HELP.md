# Getting Started
#致开云端系统
######本系统主要是通过socket连接，将机器发送的数据解析并保存到服务器上，用户通过登录系统就可查看本用户绑定的机器测量的数据
### 使用框架
#####*springboot

#####*mybatis

#####*netty（4.1.0.Final）

#####*shiro（1.4.2）

###数据库

#####*mysql （版本Mysql Server 5.5）

###服务器
#####linux服务器（版本Ubuntu18.04）

###反向代理
#####nginx服务器（版本nginx-1.16.1）

####socket连接端口：8090
####平台端口：8060

##注意点
#####机器传输数据解析必须参考协议文档，根据协议进行解析
